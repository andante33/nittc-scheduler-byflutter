# nittc-scheduler-byflutter

## **現在は開発中です**

鶴岡工業高等専門学校専用時間割アプリ<br>
Android・iOSに対応しています<br>
(Windows・Web版は開発中)<br>

## 概要
既に鶴岡工業高等専門学校専用の時間割アプリは存在しているが、iOSには対応していないことから本アプリを --開発した-- 開発している<br>

## 機能

* 時間割管理 (教科・担当教員・授業場所)
* A週・B週の自動切り替え
* 休講、補講に対応
* 時間割と紐づけた課題・予定管理
* 時間割様式の設定機能 (コマ数や授業時間等を設定できるため、高専だけでなく一般校でも使用可能です)
* 次の授業を通知

## 開発環境
* Flutter
* Dart
* VScode
