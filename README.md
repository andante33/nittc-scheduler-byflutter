# NITTC-Scheduler by Flutter

鶴岡工業高等専門学校、その他高専専用時間割・課題管理アプリ<br>
Android・~~iOS~~に対応しています。<br>
(Windows・Webは開発中です)

## 概要

鶴岡工業高等専門学校専用時間割アプリは既に存在するが、WindowsやiOSでも使えるようにするため、「Flutter」というものを使用し、新たにアプリを作成した。<br>

## 機能
* 時間割表示
* 時間割と関連付けた課題・予定管理
* A週B週表示切り替え

## 開発環境
* Flutter
* Dart
* VScode
